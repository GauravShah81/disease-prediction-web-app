# This is a Disease Prediction Web app.
<br>
This Web app is used to predict wether a person has said disease or not. Through this web app, diseases like Diabetes, Parkinson's and heart disease can be predicted.
<br>
This web app also shows the confidence percentage of the prediction given by the model.
<br>
To use Predict the disease, you have to give the required inputs and based on the given inputs, the model predicts wether the person has that said disease or not.
